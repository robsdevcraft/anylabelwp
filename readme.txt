White-label popular WordPress plugins for agencies and WAAS providers. Create a unified, branded admin experience.

== Description ==

AnylabelWP helps agencies, freelancers, and Website-as-a-Service providers create a consistent, professional admin experience by white-labeling popular WordPress plugins.

**Supported Plugins:**
* Fluent Forms (Free version)
* Fluent CRM (Free version) 
* Fluent SMTP
* WP Social Ninja (Free version)

**Key Features:**
* Replace plugin logos with custom branding
* Rename menu items for consistency
* Remove upgrade prompts and promotional content
* Clean WordPress admin interface
* Modular design - enable features per plugin
* Role-based access control

**Perfect For:**
* Website as a Service (WAAS) providers
* Digital agencies managing client sites
* Freelancers creating custom solutions
* Developers building white-label products

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/anylabelwp/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings â†’ AnylabelWP to configure
4. Install and activate supported plugins to use white-labeling features

== Frequently Asked Questions ==

= Which plugins are supported? =

Currently: Fluent Forms, Fluent CRM, Fluent SMTP, and WP Social Ninja (free versions).

= Do I need the supported plugins installed? =

Yes, AnylabelWP only white-labels plugins that are already installed and active.

= Can I use my own logos? =

Yes, you can upload custom logos for each supported plugin through the settings page.

= Does this work with pro versions? =

Currently designed for free versions, but may work with some pro features.
